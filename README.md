# DesTra-Travel-Expense-Tracker

## Overview

DesTra is a Travel Expense Management Web Application designed to help users plan trips, estimate travel expenses, monitor spending, and analyze expenses through interactive visualizations.

The application provides a simple and user-friendly interface for travelers to manage their travel budget efficiently. Users can create travel plans, calculate expenses across different categories, view graphical reports, track trip history, and access route information through interactive maps.

## Features

### User Authentication
- User Registration
- User Login
- Google Sign-In
- Forgot Password
- Secure Authentication using Firebase

### Trip Planning
- Source and Destination Selection
- Travel Duration Selection
- Hotel Preference Selection
- Currency Selection

### Expense Management
- Travel Expenses
- Food Expenses
- Shopping Expenses
- Return Expenses
- Other Expenses

### Expense Analysis
- Pie Chart Visualization
- Bar Chart Visualization
- Percentage-based Expense Breakdown
- Category-wise Expense Analysis

### Additional Features
- Currency Converter
- Interactive Maps using Leaflet.js
- Travel Route Visualization
- Trip History Tracking
- Responsive User Interface

## Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Authentication
- Firebase Authentication

### Visualization
- Chart.js

### Mapping
- Leaflet.js

## Project Workflow

1. User creates an account or logs in.
2. User enters travel details.
3. User adds travel-related expenses.
4. System calculates total expenses.
5. Expense data is displayed using charts.
6. Users can review previous trip history.
7. Currency conversion and route visualization assist in travel planning.
